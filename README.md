# 🍳 My First Web Dev Project: Recipe Hub

A foundational project showcasing my very first steps into web development. This is a purely structural, multi-page recipe website built entirely with semantic HTML5. 

Keeping this repository active serves as a personal milestone, highlighting the journey from learning basic HTML tags to currently architecting modular, full-stack web applications.

### 📖 Overview
This project is a simple collection of recipes, featuring:
- A central `index.html` navigation hub.
- Dedicated recipe pages (Lasagna, Sweet Beef, Garlic Shrimp).
- Proper use of HTML5 semantic tags (`<main>`, `<section>`, `<figure>`).
- Embedded external imagery and hyperlinks to video tutorials.

### 🚀 Getting Started
To view this project locally, no build tools are required:
1. Clone this repository to your local machine.
2. Open the folder.
3. Double-click `index.html` to open it directly in your web browser.
